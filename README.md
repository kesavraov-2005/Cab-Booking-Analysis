# Cab-Booking-Analysis
Cab Booking Analysis is a data analysis and machine learning project used to study customer booking behavior, trip patterns, ride demand, cancellations, and payment trends in cab services. The main goal of this project is to extract useful insights from cab booking data and improve business decisions.

Used SQL queries to clean, filter, join, and analyze data from multiple tables. Applied aggregate functions, GROUP BY, ORDER BY, subqueries, and window functions to identify booking trends, peak ride hours, top customers, revenue generation, and cancellation patterns.

Generated reports to analyze ride demand, driver performance, customer behavior, and payment preferences. Improved decision-making by identifying factors affecting cancellations, ride completion rates, and revenue growth.

**Key Analysis Performed**
**Total bookings and completed rides analysis
**Ride cancellation analysis
**Peak booking time identification
**Revenue and fare analysis
**Top customers and frequent riders
**Driver performance analysis
**Payment method usage analysis
**Location-wise booking trends

**SQL Concepts Used**
**SELECT, WHERE, ORDER BY
**GROUP BY and HAVING
**INNER JOIN, LEFT JOIN
**Aggregate Functions (SUM, AVG, COUNT)
**Subqueries
**CASE Statements
**Window Functions (RANK, ROW_NUMBER)
